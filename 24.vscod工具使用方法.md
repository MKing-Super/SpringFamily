# vscode使用方法

## 一、启动springboot

**扩展商店**

```bash
Extension Pack for Java（Java 扩展包）
Spring Boot Extension Pack（Spring Boot 工具包）
MyBatis Boost
```

**全局配置**

按下 `Ctrl + Shift + P`打开命令面板

输入 `Preferences: Open User Settings (JSON)`并选择

在打开的 `settings.json`文件中添加以下配置：

```json
{
    // 设置Maven可执行文件路径
    "maven.executable.path": "E:\\apache-maven-3.8.3\\bin\\mvn.cmd",
    
    // 设置Maven配置文件的路径
    "maven.settingsFile": "E:\\apache-maven-3.8.3\\conf\\settings.xml",
    
    // 设置Maven全局仓库路径（可选）
    "maven.globalSettings": "E:\\apache-maven-3.8.3\\conf\\settings.xml",
    
    // 设置本地仓库路径（可选）
    "java.configuration.maven.userSettings": "E:\\apache-maven-3.8.3\\conf\\settings.xml"
}
```



在项目下创建 .vscode 文件夹

setting.json

```json
{
    "java.home": "D:\\Java\\jdk1.8.0_181",
    "java.configuration.maven.userSettings": "E:\\apache-maven-3.8.3\\conf\\settings.xml",
    "maven.executable.path": "E:\\apache-maven-3.8.3\\bin\\mvn.cmd"
}
```

launch.json

```json
{
    "version": "0.2.0",
    "configurations": [
        {
            "type": "java",
            "name": "DemoApplication",
            "request": "launch",
            "mainClass": "com.example.demo.DemoApplication",
            "projectName": "demo"
        }
    ]
}
```

maven

```
# 2. 清理项目
mvn clean

# 3. 强制更新依赖（-U参数）
mvn clean compile -U
```
