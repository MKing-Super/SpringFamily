# Windows10常用命令

## 一、进程

```bash
# 查看所有 Java 进程及其 PID
tasklist | findstr java

# 指定进程映像名称 杀死进程
taskkill /f /im java.exe

# 查询特定 PID 的所有端口
netstat -ano | findstr "PID"

# 查找占用端口的进程
netstat -ano | findstr "端口号"
# TCP    0.0.0.0:8080           0.0.0.0:0              LISTENING       1234

# 根据 PID 杀死进程
taskkill /F /PID 1234 
```
